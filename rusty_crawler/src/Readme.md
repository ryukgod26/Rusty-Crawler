# Rusty Crawler
This is a Game Created in Rust by me. I used rltk library for the creation of this Game. This is a turn based rougelike dungeon crawler game.This Project Used Ecs (Entity Component System). This uses Procedural Dungeon Generation and field of mechanics. This Project implements a Complex Dungeon Interactions, Procedural Generation and Tactical Combat. This Project is currently in Development and the current releases only contains a Demo of the Project.

# How to Run
You can Play this game using the executables in the releases page according to your Operating System.
